使用 Giscus 为 GitHub Pages 添加评论系统

📅 2026-03-25 🏷️ 技术分享 👤 haisi-ai

Giscus 是一个基于 GitHub Discussions 的评论系统，非常适合部署在 GitHub Pages 上的静态网站。

为什么选择 Giscus？

Giscus 完全免费，无需服务器，所有评论数据都存储在 GitHub 仓库的 Discussions 中。用户使用 GitHub 账号登录即可评论，支持 Markdown、代码高亮、@提及等 GitHub 原生功能。

如何配置？

只需要在 GitHub 仓库中开启 Discussions 功能，安装 Giscus App，然后将提供的脚本代码嵌入到页面中即可。整个过程不超过 5 分钟。

代码示例

<script src="https://giscus.app/client.js"
    data-repo="haisi-ai/blogs"
    data-repo-id="R_kgDORwBnLg"
    data-category="Announcements"
    data-category-id="DIC_kwDORwBnLs4C5OQ7"
    data-mapping="pathname"
    data-strict="0"
    data-reactions-enabled="1"
    data-emit-metadata="0"
    data-input-position="bottom"
    data-theme="preferred_color_scheme"
    data-lang="zh-CN"
    crossorigin="anonymous"
    async>
</script>

现在，你可以在下方评论区留言测试了！👇
