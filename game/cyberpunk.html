《赛博朋克2077》最新DLC完整安装教程

📅 2026-03-25 👤 海斯 🏷️ 游戏教程

《赛博朋克2077》是一款由CD Projekt Red开发的开放世界角色扮演游戏。本文将详细介绍如何安装最新DLC「往日之影」。

📦 安装步骤

  1. 下载游戏本体（约70GB）
  2. 下载DLC文件（约25GB）
  3. 将DLC文件复制到游戏目录
  4. 运行游戏即可

💻 配置要求

  • 操作系统：Windows 10/11 64-bit
  • 处理器：Intel Core i7-12700或AMD Ryzen 7 7800X3D
  • 内存：16 GB RAM
  • 显卡：NVIDIA GeForce RTX 2060或AMD Radeon RX 5700
